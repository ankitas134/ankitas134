## Projects

**FinGuide — AI Investment Recommendation Engine** · *In Progress*  
Full-stack ML application targeting financial advisory for underserved users in India. Built a React frontend with a Flask REST API, ensemble models (Random Forest + Gradient Boosting) across 6 asset classes, and a 15-feature pipeline. Includes a What-If income simulator and live allocation charts.  
`Python` `Flask` `React` `XGBoost` `Scikit-learn`

---

**AI in Ayurveda — Prakriti Body Constitution Classifier** · *In Progress*  
ML web app that classifies Vata/Pitta/Kapha body types using an RF + SVM ensemble on a 13-feature questionnaire, achieving 85%+ accuracy. Delivers confidence scores, dosha charts, and personalised dietary and seasonal recommendations.  
`Python` `Streamlit` `Flask` `Scikit-learn`

---

**Multilingual NLP Converter** · *Completed (Nov 2024)*  
NLP pipeline achieving 82% accuracy across 3 language pairs using TF-IDF vectorization, POS tagging, and precision/recall evaluation.  
`Python` `NLP` `TF-IDF` `Scikit-learn`
